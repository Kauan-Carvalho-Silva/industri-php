# Sistema de Gestão Industrial

Este projeto foi desenvolvido em PHP com o objetivo de simular um sistema de gestão industrial. O sistema permite o cadastro de funcionários, produtos e ordens de produção, facilitando o controle das informações de uma indústria.

## Funcionalidades
- Cadastro de funcionários;
- Cadastro de produtos;
- Controle de ordens de produção;
- Organização do projeto utilizando Git e GitHub para controle de versão.

## Tecnologias Utilizadas
- PHP
- Git
- GitHub

## Estrutura do Projeto
